# 🚨 SMART ALERT SYSTEM

ระบบตรวจสอบสภาพแวดล้อมและแจ้งเตือนแบบเรียลไทม์

SMART ALERT SYSTEM เป็นระบบ IoT สำหรับเฝ้าระวังและติดตามสภาพแวดล้อมภายในอาคาร โดยสามารถแจ้งเตือนผู้ใช้งานได้ทันทีเมื่อเกิดเหตุการณ์ผิดปกติ เหมาะสำหรับห้องเซิร์ฟเวอร์ บ้านอัจฉริยะ ห้องทดลอง หรือพื้นที่ที่ต้องการการดูแลเป็นพิเศษ

---

## 📌 รายละเอียดโครงการ

โครงการนี้ใช้ ESP32 เชื่อมต่อกับเซนเซอร์ต่าง ๆ เพื่อเก็บข้อมูล เช่น อุณหภูมิ ความชื้น และระดับควัน จากนั้นจะแสดงผลผ่านหน้า Dashboard และส่งการแจ้งเตือนไปยัง Discord แบบเรียลไทม์ เมื่อค่าที่ตรวจพบเกินกว่าระดับที่กำหนดไว้

## 🛠️ เทคโนโลยีที่ใช้

### Hardware

- ESP32
- เซนเซอร์ DHT11 / DHT22
- เซนเซอร์ตรวจจับควันหรือแก๊ส MQ-2
- Relay Module

### Software

- HTML
- CSS
- JavaScript
- Google Apps Script
- Discord Webhook
- Discord Bot
- Netlify

## 🚀 วิธีการใช้งาน

### 1. Clone โปรเจกต์

```bash
git clone https://github.com/Tirawit01/SMART-ALERT-SYSTEM.git
```

### 2. ตั้งค่าระบบ

แก้ไขค่าการเชื่อมต่อให้ตรงกับการใช้งานของคุณ

```javascript

```

### 3. Deploy Dashboard

นำส่วน Dashboard ไป Deploy ผ่าน Netlify

### 4. อัปโหลดโค้ดลง ESP32

อัปโหลดเฟิร์มแวร์ผ่านโปรแกรม Arduino IDE
